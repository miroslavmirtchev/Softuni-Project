
# Softuni Project

This is StyleSphere, my final year project for Softuni Buditel


## How To Run It Locally

Clone the project

```bash
  git clone https://github.com/miroslavmirtchev/Softuni-Project.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon server.js
```
